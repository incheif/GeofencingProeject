Real-Time Locations

View the real-time locations of all tracked devices on an interactive map. This page provides up-to-date information on where each device is currently situated.

Historical Data

Access historical location data for all devices. This page allows you to see where devices have been over time, helping you track movement patterns and past locations.

Mapped History

Visualize the movement history of devices on a map. This feature plots the historical locations of devices, allowing you to see travel routes and frequently visited areas.

Geofencing (Testing)

Test and experiment with geofencing features. Set up virtual boundaries and receive alerts when devices enter or exit these predefined areas. This is a sandbox environment for testing purposes.

Geofencing (Restricted Access)

Access the production version of the geofencing feature. This page is restricted to authorized users and allows for setting up and managing geofences in a live environment.

Define Custom Region

Define and add custom geographical regions to the system. These regions can be used for various purposes such as geofencing, area-based reporting, or location-specific alerts.
